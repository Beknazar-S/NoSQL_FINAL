# Football Management System (Assignment 3 Part 2)

## Deployed URL
**Link:** https://asiq3p2.onrender.com

## Features
- **Create**: Add new football items/players via web form.
- **Read**: View all records in a dynamic list.
- **Update**: Edit existing records using pop-up prompts.
- **Delete**: Remove records from MongoDB.

## Tech Stack
- **Backend**: Node.js, Express
- **Database**: MongoDB Atlas
- **Frontend**: Vanilla HTML, CSS, JavaScript (Fetch API)

## Setup Instructions
1. Clone the repository.
2. Run `npm install`.
3. Create a `.env` file with `MONGO_URI` and `PORT`.
4. Run `npm start` to launch the server.
